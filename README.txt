-- SUMMARY --

This Autocomplete flag select module enables a user 
to tag another user easily via an autocomplete search textbox. 

-- REQUIREMENTS --

None.


-- INSTALLATION --

* Install as usual, see http://drupal.org/node/70151 for further information.


-- CONFIGURATION --

* Configure user permissions in Administer >> User management >> Permissions >>
  Autocomplete flag select module:

  - adiminister get satisfaction

    Users in roles with the "allow autocomplete flag select access" permission will be 
    able to configure the module site-wide.

* Customize the settings in Administer >> Site building >> 
  Autocomplete flag select.
